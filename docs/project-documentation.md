## Installation

1. Clone or download this project
2. Open `index.html` in a web browser
3. No additional setup required - all dependencies are loaded via CDN

## Usage

- View real-time environmental metrics on the main dashboard
- Interact with charts to see historical trends
- Use the global map selector to view different environmental metrics
- Explore personalized recommendations for sustainability improvements
- Join community initiatives and events

## Technologies Used

- **HTML5**: Semantic structure and accessibility
- **CSS3**: Modern styling with CSS Grid and Flexbox
- **JavaScript**: Interactive functionality and data handling
- **Chart.js**: Data visualization and real-time charts
- **Font Awesome**: Icons and visual elements

## Browser Compatibility

- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+
